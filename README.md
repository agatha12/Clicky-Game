Clicky Game
========== 

Overview Description
--------------------

Clicky Game is a memory game where users must click on 12 unique images and try not to click the same one twice. 


Installation
-----------------

- Run "yarn install" 
- Run "yarn start"


Skills Used
-----------

- React
- HTML 
- CSS / Bootstrap
- Javascript
- Heroku
